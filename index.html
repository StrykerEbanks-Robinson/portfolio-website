Hello!

Tic-Tac-Toe (AI)

This was one of the first projects that I completed. Using Python, I built a Tic-Tac-Toe game where you can play against the computer or another human. If you pick the computer, you can also choose from three levels of difficulty, where Medium and Hard use a minimax algorithm so that the computer can play intelligently. Feel free to check out the code for my tic-tac-toe game on my GitHub!

Minesweeper

Since my fiancé really enjoys minesweeper, I built her a minesweeper game as one of my first Python projects. While the minimax algorithm in my tic-tac-toe project helped me to better understand recursive functions, this minesweeper project gave me a much firmer grasp of Python classes and the strengths of OOP.
